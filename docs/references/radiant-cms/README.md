## Notes

The `wiki/` directory is the most useful place to inspect Radiant's historical content model, including pages, layouts, snippets, Radius tags, extensions, and authentication-related notes.

The `source/` directory is preserved for implementation details and terminology. Treat it as an external reference snapshot, not application code for Evolve.
